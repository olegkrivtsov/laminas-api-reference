Class AdapterPluginManager

Summary

Fully Qualified Name: Laminas\Paginator\AdapterPluginManager
Extends: AbstractPluginManager

Description

Plugin manager implementation for paginator adapters.

Enforces that adapters retrieved are instances of Adapter\AdapterInterface. Additionally, it registers a number of default adapters available.
